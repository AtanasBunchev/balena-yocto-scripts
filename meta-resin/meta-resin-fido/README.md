# Resin.io layer for fido supported boards

## Description
This repository enables building resin.io for fido supported machines.

## Supported machines
raspberrypi
raspberrypi2
parallella-hdmi-resin
nitrogen6x
nuc
cubox-i
beaglebone

## Layer dependencies

This layer depends on:

* URI: git://git.yoctoproject.org/poky
    * branch: fido
